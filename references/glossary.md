# Glossary — 온라인 광고 용어 60+

인하우스 마케터 기준 자주 쓰는 용어. 한국어 의미 + 영어 원문 + 1~2줄 설명 + 관련 용어.

## 1. 비용·효율 지표

| 약어 | 원문 / 한국어 | 정의 | 공식·예시 | 관련 |
|---|---|---|---|---|
| **CPM** | Cost Per Mille / 노출 1000회당 비용 | 브랜드 광고·Reach 캠페인 기준 | 총비용 ÷ 노출 × 1000 | CPC, vCPM |
| **CPC** | Cost Per Click / 클릭당 비용 | 트래픽·검색광고 기준 | 총비용 ÷ 클릭 | CPM, CPA |
| **CPA** | Cost Per Acquisition / 전환당 비용 | 전환(가입·구매)당 광고비 | 총비용 ÷ 전환수 | CAC, tCPA |
| **CPI** | Cost Per Install / 앱 설치당 비용 | 앱 다운로드 캠페인 전용 | 총비용 ÷ 설치수 | CPA, tCPI |
| **CAC** | Customer Acquisition Cost / 고객 획득 비용 | 광고비 + 운영비 포함 종합 | 총마케팅비 ÷ 신규고객 | CPA, LTV |
| **LTV** | Lifetime Value / 고객 생애 가치 | 고객 1명이 생애에 지출하는 총액 | ARPU × 평균 체류기간 | CAC, 수익성 지표 |
| **ROAS** | Return On Ad Spend / 광고 지출 대비 매출 | 100% = 본전, 300% = 3배 회수 | (매출 ÷ 광고비) × 100% | ROI, tROAS |
| **ROI** | Return On Investment / 투자 대비 수익률 | ROAS와 유사하나 운영비 포함 | ((수익-비용) ÷ 비용) × 100% | ROAS |
| **tCPA** | Target Cost Per Action / 목표 CPA | Smart Bidding에서 설정한 목표값 | 사용자가 지정 | CPA, Cost Cap |
| **tROAS** | Target Return On Ad Spend / 목표 ROAS | Smart Bidding에서 설정한 수익률 목표 | 사용자가 지정 (예: 3.0 = 3배) | ROAS, Value Optimization |

## 2. 퍼널·행동 지표

| 약어 | 원문 / 한국어 | 정의 | 공식·예시 | 관련 |
|---|---|---|---|---|
| **CTR** | Click-Through Rate / 클릭률 | 노출 중 클릭한 비율 | (클릭 ÷ 노출) × 100% | CVR, Engagement Rate |
| **CVR** | Conversion Rate / 전환율 | 클릭 중 구매/가입한 비율 | (전환 ÷ 클릭) × 100% | CTR, ROAS |
| **Hook Rate** | 3초 시청 통과율 | 3초 이상 재생한 사용자 비율 (TikTok·Meta) | (3초 통과자 ÷ 노출) × 100% | CTR, Engagement |
| **Hold Rate** | 콘텐츠 완청률 | 영상 전체를 본 사용자 비율 | (완청자 ÷ 노출) × 100% | Hook Rate |
| **Engagement Rate** | 상호작용률 | 좋아요·공유·댓글 등 모든 상호작용 | (상호작용 ÷ 노출) × 100% | CTR, Hook Rate |
| **Thumb-Stop** | 스크롤 정지율 | 사용자가 스크롤을 멈춘 비율 (주로 모바일) | 플랫폼별로 정의 다름 | Hook Rate |
| **View Rate** | 조회율 | 노출된 광고 중 시작한 비율 | (영상 시작 ÷ 노출) × 100% | CTR |

| **Value Optimization** | Value Optimization / 가치 기반 최적화 | AOV/LTV 기반 입찰 (수량 아닌 가치 중심) | Meta, Google | Max Conversion Value |

| **tCPA** | Target Cost Per Action / 목표 CPA | 기계학습으로 목표 CPA 달성하도록 자동 입찰 | Meta, Google | Smart Bidding |
| **tROAS** | Target Return On Ad Spend / 목표 수익률 | 기계학습으로 목표 ROAS 달성하도록 자동 입찰 | Meta, Google | Value Optimization |
| **Max Conversions** | Max Conversions / 전환 최대화 | 예산 내 전환수를 극대화하는 자동 입찰 | Google | Cost Cap |
| **Max Conversion Value** | Max Conversion Value / 전환값 최대화 | 전환수 대신 전환의 총 가치를 극대화 | Google, Meta | Value Optimization |
