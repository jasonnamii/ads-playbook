기준일: 2026-04-22. 다음 업데이트: 2026-07-01 예정 (Q3 초).

## Meta (Andromeda 시대)

### 1. Andromeda 엔진 업데이트
**변경**: Meta의 핵심 AI 엔진 전면 강화. 입찰, Audience 최적화, 크리에이티브 선택까지 통합.

**구체 사항**:
- Advantage+ 자동화 범위 확대 (Placement·Audience·Budget 동시 최적화)
- Learning Phase 입장 기준 완화 (50→30 conversions 검토 중)
- Generative AI 광고 생성 표준화 (Q4 2026까지)

**인하우스 대응**:
- 크리에이티브 파이프라인: 주 1회 신규 자산 투입 (AI 생성 병행)
- CBO 신뢰: ABO 사용 극소화 (알고리즘 학습 방해)
- 브랜드 보호: Advantage+ 캠페인에서 Brand Exclusion 필수

---

### 2. Generative AI Ads 확대
**변경**: AI 기반 광고 자동 생성·추천 메인 메커니즘화.

**주요 기능**:
- Text-to-Image: 텍스트 프롬프트 → 고품질 정적 이미지
- Text-to-Video: 텍스트 → 5~15초 브랜드 비디오
- Background Generation: 제품 사진에 배경 자동 생성
- Voice & Audio: 텍스트 → 음성 나레이션

**제약 & 의무**:
- AI 생성/수정 콘텐츠 공시 필수 (April 2026부터 강화, 미공시 시 광고 거절 증가)
- High-profile brand는 여전히 수동 검증 권장 (신뢰도 이유)

**인하우스 대응**:
- AI 도구 활용: Midjourney·DALL-E·Canva AI 등 조기 도입
- 창의성 부족 극복: AI 생성 → 수동 보정 프로세스
- 글로벌 규제 추적: 각 국가별 AI 콘텐츠 규제 동향 모니터링

---

### 3. Attribution Window 단축
**변경**: iOS 귀속 기간 7일(CTC) → 1일로 축소 검토 중.

**배경**:
- Apple SKAN 4.0의 지연(24~100일)로 인한 재귀속 필요
- Meta AEM 2.0 도입으로 후기 전환 복구 강화

**현황 (2026년 4월)**:
- CTC 7일: 유지 중 (공식 변경 예정 없음)
- View-through 1일: 예상대로 적용
- 영향: 초기 성과 보고 더 엄격 (첫 24h 신호 중요)

**인하우스 대응**:
- 러닝 페이즈: "50 conversions"를 3~5일 기준으로 재평가
- 어트리뷰션 모델: 7일 CTC에서 1일로 커스터마이즈 (선택 가능)
- 측정 리프레시: 48시간~72시간 단위 보고

### 4. Advantage+ Creative 통합
**변경**: Meta의 동적 크리에이티브 최적화 표준화. 모든 캠페인에 권장 (필수 아님).

**기능**:
- 업로드된 자산 (이미지·비디오·텍스트) 조합 자동 테스트
- AI 기반 성과 예측 ("이 조합은 CTR 35% 예상")
- 브랜드 규격 자동 추출 (로고, 색상, 폰트)

**성과**:
- CTR 30% ↑, 전환율 20% ↑ (기준 수동 최적화 대비)
- 크리에이티브 피로 50% 감소

**인하우스 대응**:
- 자산 다양성: 최소 15개 이미지 + 5개 비디오 (Advantage+ 권장)
- 수동 A/B: 여전히 필요 (최고 성능 검증용)
- 브랜드 가이드: AI 생성 이미지의 톤 일관성 체크

---

### 5. AI 콘텐츠 공시 의무화
**변경**: 모든 AI 생성/수정 콘텐츠는 "AI-generated", "AI-edited" 라벨 부착 필수.

**규제 배경**:
- 미국 FTC, EU, 한국 등 글로벌 규정 수렴
- Meta March 2026부터 자동 감지·라벨 부착 시작

**미공시 시 결과**:
- Ads Review 거절 (10% 증가)
- 어카운트 경고 (누적 시 제약)

**인하우스 대응**:
- 콘텐츠 출처 추적: AI 생성 여부를 크리에이티브 메타데이터에 기록
- 정책 준수: AI 도구 사용 시 공식 라벨 활용 (Canva, Midjourney 내장 옵션)

### 2. Customer Match API 마이그레이션 (2026-04-01 시행)
**변경**: Google Ads API를 통한 Customer Match 신규 업로드 **종료**. Data Manager API로 강제 이관.

**기한**: 2026-04-01 완료 필수

**영향 범위**:
- API 기반 자동화 (스크립트·구글 마케팅 플랫폼)
- 수동 CSV 업로드는 영향 없음

**필요 조치**:
```
Google Ads API 대신:
→ Google Data Manager API로 마이그레이션
→ 또는 UI에서 수동 업로드 유지
```

**인하우스 대응**:
- 기술팀 검증: 자동화 스크립트가 Ads API 사용 중인지 확인
- 새 API 테스트: Data Manager API의 권한·속도 미리 검증
- 혼합 전략: ActiveCampaign·HubSpot 커넥터로 자동 동기화 (Ads UI 내 권장)

### 3. Broad Match + Smart Bidding 권장 강화
**변경**: Google이 정확 매칭 완전 폐지가 아닌, "Broad + Smart Bidding은 25% 전환 ↑"로 데이터 공시.

**성과 데이터** (2026년 Q1):
- Broad Match + tCPA: 전환 +25% (tCPA 유지)
- Exact Match 고집: 도달 범위 -40%, 기회 손실

**Google 메시지**:
- Exact는 여전히 "브랜드 수호용"
- Broad는 "신규·확장용" (보수적 Negative Keywords 필수)

**인하우스 대응**:
- 검색 캠페인 구조: Search 상단(Exact) + 중위(Phrase) + 하위(Broad + Smart)
- 시험적 전환: 상위 성과 Exact Ad Set를 Broad로 복제 → 7일 비교
- Negative Keywords: Master List 강화 (직업, 학술, 무료 등)

---

### 4. DDA (Data-Driven Attribution) 기본화
**변경**: 모든 신규 Google Analytics 4 계정에서 DDA가 기본 귀속 모델로 설정됨.

**요구사항**:
- 최소 200 conversions / 30일
- 2,000 interactions (모든 전환 누적)

**변경 의미**:
- Last-Click (과거 표준) 폐지 추세 가속화
- UI에서 수동 선택 불가 (자동 설정만)

**인하우스 대응**:
- GA4 검증: DDA 활성화 여부 확인 (Status: "Enabled" 표시)
- 어트리뷰션 재해석: Last-Click 리포트 폐기 (DDA 결과만 참조)
- 고급 분석: "Conversion Attribution Analysis" (2월 베타)로 Assisted Conversions 분석

---

### 5. Gemini 광고 확대 (실험 중)
**변경**: Google AI Overviews 내 광고, Gemini 앱 내 광고 확대.

**현황**:

**인하우스 대응**:

## TikTok

### 1. Smart+ 자동화: All-or-nothing 폐지
**변경**: Smart+의 "모든 걸 자동 또는 모든 걸 수동" 규칙 폐지. **부분 자동화** 선택 가능.

**신규 옵션**:
- Targeting: 자동 / Audience: 수동
- Creative: 자동 / Budget: 수동 (가드레일)
- 또는 전체 수동

**이점**:
- 크리에이티브 다양성 필요한 브랜드: 자동 선택
- 정밀한 예산 제어: 가드레일만 설정

**인하우스 대응**:
- 성숙도 높은 계정: 크리에이티브만 자동, 예산은 CAP 설정
- 초기 계정: 전체 자동 → 2주 후 예산 수동 전환

---

### 2. Symphony AI 생산화
**변경**: Symphony (TikTok 생성형 AI) 완전 운영. Text-to-Video 주요 기능.

**기능**:
- Dreamina 모델: 텍스트 프롬프트 → 5~15초 고품질 비디오
- Image-to-Video: 스틸 이미지 → 동적 비디오
- Showcase Products: 상품 카탈로그 → 자동 비디오 생성

**성과**:
- 크리에이티브 제작 비용 60~70% 감소 (UGC 외주 대체)
- 로테이션 속도 2배 (신규 크리에이티브 1주일 단위)

**인하우스 대응**:
- Symphony 도입: 초기 테스트 시작 (UGC 클론 대체 목적)
- 브랜드 톤: AI 생성 이미지는 "원본 + 수정" 2-step 워크플로우
- 비용 절감: 월간 UGC 외주 예산 30~40% 감소 기대

---

### 3. Shop Ads 글로벌 확대
**변경**: TikTok Shop Ads (이커머스) 글로벌 표준화. VSA와 통합 진행.

**현황**:
- 미국·영국·일본·한국 등 15개국 가용
- VSA와 Shop Ads 점진적 통합 (6월까지)

**영향**:
- Catalog Ads = VSA + Shop Ads 단일 포맷화
- CVR 2배 성과 확장

**인하우스 대응**:
- Shop Ads 테스트: 카탈로그 미노출 지역에서 우선 실행
- 성과 기댓값: Web 링크 대비 CPA 20~40% 개선
- 정산 구조: Shop Ads는 TikTok 직결제로 (제3자 링크 대비 마진 개선)

---

### 4. Video Shopping Ads (VSA) 단위 강화
**변경**: VSA의 동적 상품 카드, AI 개인화 알고리즘 업그레이드.

**신규 특징**:
- Cross-product Recommendations: 광고 내 관련 상품 자동 제시
- Behavioral Personalization: 사용자 선호 기반 상품 정렬
- Live Commerce Integration: 라이브 쇼핑과 VSA 연계

**성과**:
- 전환율 (CVR) +25~35% vs. In-Feed Ads
- AOV (평균주문액) +18% (관련상품 노출)

**인하우스 대응**:
- 상품 피드: 정기적 업데이트 (월 1회 이상, 인벤토리 동기화)
- 캠페인 구조: VSA만 별도 캠페인 (최적화 신호 명확화)

---

### 5. US 규제 변동 영향
**변경**: TikTok 미국 재운영 후 정책 강화. 광고 수동 검수 재개, AI 콘텐츠 공시 의무화.

**인하우스 대응**:

- 콘텐츠 가이드: AI 생성물은 사전에 "AI" 마킹
- 정책 모니터링: TikTok 공식 정책 변경 주간 확인

## Apple Search Ads (ASA)

### 1. SKAN 완전 통합
**변경**: ASA 모든 계정에서 SKAN 4.0 필수 설정 (선택 아님).

**기한**: 2026년 4월부터 완료 (기존 IDFA-only 카운트 중단)

**영향**:
- 어트리뷰션 지연: 24~100일 (기존 실시간 대비)
- 데이터 정확도: 애그리게이션 (개별 사용자 불가)

**인하우스 대응**:
- SKAN 학습: Postback 3개 (0/1/2) 구조 파악
- MMP 연동: AppsFlyer·Adjust와 SKAN 데이터 통합
- 어트리뷰션 재평가: 지연을 감안한 성과 판단 기준 (최소 14일 이상)

---

### 2. Maximize Conversions 입찰 자동화 (신규)
**변경**: 수동 입찰 대신 "Maximize Conversions" 자동화 권장.

**특징**:
- 캠페인 레벨 CPA 목표 설정 → 시스템이 동적 CPM 조정
- Apple의 온디바이스 머신러닝 (프라이버시 준수)

**기대**:
- CPA 안정화 (±15% 변동)
- 도달 범위 +20~30% (자동 최적화)

**인하우스 대응**:
- 초기 단계: Maximize Conversions로 시작 (4주 데이터 수집)
- 안정화 후: 세분 제어 필요시 수동 입찰로 전환
- 목표 CPA: 과거 3개월 평균 ± 10% 수준으로 설정

---

### 3. Custom Product Pages (CPP) 필수화
**변경**: 모든 고성과 캠페인에 CPP (앱 스토어 맞춤 페이지) 사실상 필수.

**성과 데이터**:
- 게임: 전환율 +8%
- 비게임: 전환율 +6.6%
- 전체 평균: +7.2%

**권장 구조**:
- 키워드 그룹별 1개 CPP (예: "fitness" → "운동" 테마 페이지)
- 각 CPP당 2~3개 스크린샷 순서 A/B 테스트

**인하우스 대응**:
- CPP 로드맵: 월간 2~3개 신규 생성 (지속적 커버리지 확대)
- 콘텐츠 개선: 스크린샷 1번째가 전환에 40% 영향 (최우선 최적화)
- 테스트 기간: 각 CPP 당 최소 14일 (통계 신뢰도 확보)

---

### 4. 다중 슬롯 확장
**변경**: ASA 검색 결과 내 복수 광고 위치 가능 (2026년 3월 롤아웃).

**구조**:
- 기존: 검색 결과당 1개 광고 위치
- 현재: 최대 2~3개 위치 가능

**CPM 영향**:
- 경합 심화 → CPM +10~20% 예상
- 또는 경합 회피 → 하위 슬롯 저가 활용 기회

**인하우스 대응**:
- 예산 조정: CPM 상승 대비 +15% 프로비전
- 다중 광고: 같은 키워드에 2개 광고 제출 (테스트)

## Naver

### 1. 성과형DA 업데이트 & AI 추천 강화
**변경**: GFA 자동화 알고리즘 고도화. 초기 계정도 20+ conversions 후 자동 학습 가능.

**신기능**:
- AI 자동 진단: 신규 광고주 계정을 분석하여 최적 구조 제안
- Cross-category Learning: 유사 업체 데이터로 초기 입찰 최적화

**성과 개선**:
- ROAS 3.0~5.0 달성 사례 증가
- 러닝 기간 단축 (4주 → 2~3주)

**인하우스 대응**:
