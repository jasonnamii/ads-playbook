# platform_naver_kakao — Naver & Kakao 광고 운영 가이드

인하우스 마케터가 Naver(검색/쇼핑/GFA)와 Kakao(모먼트/메시지)를 운영할 때 핵심.

## 개요

**한국 이커머스의 표준 채널.** Naver는 검색량(월 7.3억), 쇼핑 플랫폼(스마트스토어), 로컬(플레이스) 강화. Kakao는 CRM(메시지 80% 열람율), 낮은 CPM, 콘텐츠형 마케팅. 최적 조합: Naver 60% + Kakao 40% (이커머스 기준).

**언제 쓰나:** 한국 이커머스/서비스 (Naver), 재구매/CRM (Kakao 메시지), 뉘치 검색 (Daum).

**언제 피하나:** Meta/Google이 2배 이상 효율 높으면 해외 고객군 우선.

## Naver 광고 구조 3가지

### 1. 검색광고 (파워링크 중심)

**포맷:**
- **파워링크:** 검색결과 상단 (클릭당 CPC 기반)
- **브랜드검색:** 자사 브랜드 검색 시 최상단 (가장 저가)
- **쇼핑검색:** 상품명 매칭 (상품 이커머스 전용)
- **플레이스광고:** 지역·업종 실명 (카페, 병원, 부동산 등)

**최적 입찰 구조:**
- 초기: 시장 평균 CPA의 70% → 데이터 수집 → 점진적 상향
- 브랜드는 10배 낮은 CPC
- 카테고리별 차등: 음식료(낮음) < 의류(중) < 전자제품(높음)

### 2. 성과형 디스플레이 (GFA)

**GFA (Glad For Advertiser) 정의:**
- RTB 기반 자동 입찰
- 목표 CPA/ROAS 설정 → AI 최적화
- Naver·Line·카카오스타일 네트워크

**최소 조건:**
- 최근 30일 15–20건 전환 필수 (알고리즘 학습)
- 일 예산 3만원 이상 권장
- 초기 낮은 CPA → 안정화 후 상향

**성공 사례:** 이커머스 ROAS 3.0–5.0 달성.

### 3. 보장형 디스플레이 (순보증)

**특징:**
- CPM 기반 (노출 수 보증)
- 브랜드 인지도, 신제품 런칭 초기
- 일반 GFA 대비 신뢰도 높음

## Naver 캠페인 목적별 구현

### 이커머스 (가장 일반적)
- **구조:** 쇼핑검색 60% + GFA 40%
- **핵심:** 상품명 최적화 (브랜드+모델+사이즈+기능)
- **입찰:** 매출 × 목표 ROAS 역산으로 최대 입찰가 설정
- **메트릭:** ROAS (매출÷광고비)
- **픽셀:** wcs.trans 기반 전환추적 (중복 제거 필수)

### 앱 다운로드
- **제약:** 직접 앱 설치 추적 불가 (정책상)
- **우회:** 앱스토어 랜딩 → 다운로드 유도
- **측정:** 앱 내 회원가입/과금 이벤트로 추적
- **연동:** Firebase/Adjust + 네이버 픽셀

### 리드 수집
- **주 매체:** 파워링크 + 지역소상공인광고
- **타겟:** 지역+검색 의도 (예: "피부과 + 00구")
- **메트릭:** CPA (상담 예약, 폼 제출)

### 리타겟팅
- **대상:** 180일 방문자, 장바구니 미완료자
- **매체:** GFA (NID 기반 오디언스)
- **메시지:** 할인, 시간 제한, 상품 추천

### 오프라인 방문 (Location)
- **매체:** 플레이스광고, 지역 검색
- **측정:** 지도 조회, 전화 클릭, 길찾기
- **활용:** 가게, 식당, 부동산

## Naver GFA 실전 운영 4단계

### 1주: 초기 세팅 (데이터 수집)
- 목표 선택: "전환수 최대화"
- CPA 제한 없음 (학습 우선)
- 일 예산: 충분함 (최소 3만원)

### 2주: 안정화 확인
- 최소 20건 전환 달성
- 평균 CPA 계산
- "CPA 목표 설정"으로 전환

### 3주: 목표 CPA 설정
- 목표 = 현재 평균 CPA × 0.8–0.9
- 입찰가 자동 최적화

### 4주+: 고급 운영
- 세그먼트별 차등 입찰 (재구매자 +30%, 신규 -20%)
- 크리에이티브 로테이션 (3개 → 2주마다 1개 교체)
- 시즈널 예산 조정

- 30초 이상 세션만 카운트

## Kakao 모먼트 상품군 (2026년)

### 1. 비즈보드 (Bizboard)
- **위치:** 카카오톡 "채팅" 탭 배너
- **특징:** 메가 트래픽 (일 70% 이상 노출)
- **단가:** CPC 300–600원 (Naver 대비 20–40% 저가)
- **운영:** 3개 배너 테스트 → 상위 2개 유지, 1개 교체 (2주마다)

### 2. 디스플레이 (카카오 네트워크)
- **범위:** 다음, 카카오맵, 카카오스토리 전체
- **입찰:** GFA 유사 성과형
- **CPM:** 1,500–3,000원 (저가)

### 3. 메시지 광고 (Message Ads)
- **열람율:** 80% 이상 (이메일 대비 5배)
- **포맷:** 친구톡(프로모션), 알림톡(주문), 오픈톡(커뮤니티)
- **성공 케이스:**
  - 구매 감사 + 재구매 쿠폰: 재구매율 35% ↑
  - 장바구니 이탈 + 시간 제한 할인: 복구 전환 28% ↑
  - 신규 회원 + 첫 구매 쿠폰: 첫 구매율 42% ↑

### 4. 카카오톡 채널 (구독 기반)
- **정의:** 직접 광고 아님 (CRM 매체)
- **메커니즘:** 친구 추가 → 메시지 발송 권한
- **활용:** 신상품 런칭, 이벤트 공지, 세일 알림

### 5. 다음 쇼핑광고
- **위치:** 다음 발견/뉴스/랭킹 8개 탭 + PC 우측
- **입찰:** CPM (일반 디스플레이 대비 5–10% 프리미엄)

### 6. 검색광고 (Daum Search)
- **특징:** Naver 대비 20–50% 저가 입찰
- **검색량:** 월 1억건 (Naver 대비 1/7)
- **활용:** 뉘치 키워드, 저가 진입 전략

## Naver vs Kakao: 채널별 최적 전략

| | Naver | Kakao |
|---|---|---|
| **강점** | 검색량, 쇼핑, 로컬 | CRM, 낮은 CPM, 메시지 |
| **예산 배분** | 60% | 40% |
| **입찰** | CPC/CPM | CPM/CPA |
| **주 메트릭** | ROAS | 재구매율, 메시지 열람 |
| **성숙도** | 성숙 (포화 경향) | 성장 (기회) |

## Kakao 메시지 광고 실전 노하우

### 메시지 성공 공식
**메시지 구조:** [개인화명] + [긴급성/혜택] + [CTA 버튼]

**예시:**
```
[이름]님, 오늘 자정까지 10% 할인!
방금 담으신 상품이 남아있습니다.

[지금 구매하기 버튼]
```

### 발송 시간 최적화
- **이커머스:** 점심 12–13시, 저녁 18–20시
- **앱:** 아침 8시, 저녁 20–21시
- **B2B:** 업무 시간 9–11시, 14–16시

### 개인화 동적 삽입
- 사용자명, 상품명, 할인율 동적 변수 활용
- A/B 테스트: 링크 없음 vs "지금 보기" 버튼 → 버튼 CTR 3배

## 인하우스 실전 노하우 (7가지)

### 1. 상품명이 모든 것 (Naver)
- 브랜드명 + 모델 + 사이즈 + 색상 + 기능
- 검색어 포함 여부 = 노출 여부

### 2. 초기 낮게, 점진적 상향 (Naver GFA)
- 첫주: CPA 제한 없음 (학습)
- 2주: 평균 CPA × 0.8로 설정
- 3주+: 세그먼트별 차등 입찰

### 3. Kakao 메시지는 CRM이다
- 광고비 아님 (구독자 관리 투자)
- 80% 열람율 = 다른 매체의 10배 신뢰
- 메시지당 ROI 5배 이상 기대

### 4. 픽셀 품질이 성과를 좌우
- 중복 제거 자동화
- 제휴사 전환 필터링
- 월 1회 데이터 정확도 감시

### 5. 시즈널 준비 2주 전
- 쇼핑시즌/명절 2주 전 예산 +30%
- 크리에이티브 사전 준비 (Kakao 콘텐츠)

### 6. Daum은 저가 진입 매체
- Naver 포화 후 추가 채널
- 동일 키워드 20–50% 저가

### 7. 재구매가 가장 효율 높음
- Naver GFA Retargeting + Kakao 메시지 조합
- 신규 대비 ROAS 5배, CPA 1/3
