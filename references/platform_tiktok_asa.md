# platform_tiktok_asa — TikTok & Apple Search Ads 운영 가이드

인하우스 마케터가 TikTok & ASA(Apple Search Ads)를 운영할 때 핵심.

## 개요

**TikTok:** 의도 창조(Intent Creation) 플랫폼 - 사용자가 원하는 걸 몰랐을 때 발견시키는 강력함. 상단 퍼널(인지/고려) 최고. **ASA:** 의도 포착(Intent Capture) - App Store 검색자 타게팅, 하단 퍼널(전환) 최고. 함께 운영하면 Full-funnel 효과.

**언제 쓰나:**
- TikTok: 신제품 런칭, 신규 고객 확보, 젊은 세대 도달, 상단 퍼널
- ASA: Brand Defense, 고효율 앱 다운로드, 경쟁사 입찰

**언제 피하나:**
- TikTok: 고가/B2B 상품(성인, B2B마케팅 플랫폼 우선)
- ASA: 브랜드 인지만 목표면 낭비

| 목표 | 매칭 | 최소 예산 | 신호 필수 | 러닝 기준 |
|------|------|---------|----------|----------|
| **Reach** (도달) | 인지만 필요 | $200–500/월 | Impression | CPM 최저 |

## TikTok 광고 포맷 5가지

### 1. In-Feed Ads (표준)
- **배치:** 피드 스크롤 중 자연스럽게 삽입
- **길이:** 15–60초 (권장 15–30초)
- **포맷:** 9:16 수직
- **성과:** 가장 높은 ROI (모든 목표에 최적)
- **크리에이티브:** 네이티브 UGC 톤 필수

### 2. Spark Ads (프리미엄 in-feed)
- **정의:** 유기 TikTok 포스트를 광고로 프로모션 (유기 상호작용 유지)
- **성과:** 표준 in-feed 대비 30% 높은 완료율, 142% 높은 참여율
- **특징:** 좋아요/공유/댓글/팔로우 모두 카운트됨 (브랜드 신뢰도 ↑)
- **실전:** 인하우스 또는 Creator Marketplace로 사전 검증된 포스트만 부스트

### 3. TopView (전면 상단)
- **배치:** 앱 오픈 시 전면 풀스크린
- **길이:** 5–60초 (권장 15–30초)
- **Skip:** 3초 후 가능
- **성과:** 광고 회상률 최고 (72% vs 30% in-feed)
- **비용:** 매우 높음 (예비 예산 시에만)
- **실전:** 신제품 런칭, 대형 프로모션, 풀-펀널의 인지 앵커 역할

### 4. Brand Takeover (스킵 불가)
- **배치:** 앱 오픈 시 즉시 3–5초 풀스크린
- **특징:** 상호작용 없음 (단순 노출)
- **빈도:** 사용자당 하루 1회 최대 (TikTok 자동 규제)
- **용도:** 극도로 단순 브랜드 메시지만 (1줄 문구)
- **주의:** ROI 측정 어려움 (상단 퍼널만)

### 5. Video Shopping Ads / Catalog Ads
- **정의:** 동적 상품 카드를 비디오 내에 삽입
- **구매 흐름:** 클릭 → 상품 슬라이드 → TikTok 앱 내 구매 (외부 사이트 이동 X)
- **성과:** 웹사이트 트래픽 대비 **CVR 2배**, In-Feed 대비 **CPA 20–40% 개선**
- **카탈로그:** TikTok Shop 또는 Shopify 통합
- **이커머스 중심:** 70% 이상 매출을 VSA에서 기대

## TikTok 크리에이티브 3가지 필수 원칙

### 1. 3초 훅 법칙
- **첫 3초가 시청 지속 판단 71% 결정**
- **4가지 훅 타입:**
  - Visual: 급격한 줌, 클로즈업, 전환
  - Text: 오버레이 텍스트 (5–10단어/초)
  - Question: "이거 알아?" 호기심
  - Emotional: 얼굴 표정 감정 전달

### 2. 네이티브 톤 (Very Important)
- **Meta와의 차이:**
  - Meta: 중상 수준 폴리시 OK, 주간 수명
  - TikTok: 극도로 "광고 같지 않은" 톤 필수
  - Meta 재사용 금지 (성과 90% 하락)
- **피하기:** 과다 자막, 브랜디드 음악, 완벽한 광고 느낌
- **해결:** Creator Marketplace 또는 UGC 제작

### 3. 크리에이티브 로테이션
- **Facebook:** 4주마다 교체
- **TikTok:** 7–10일마다 신규 자산 (빠른 피로)
- **목표:** 주 3–5개 신규 크리에이티브

## Smart+ 자동화 (TikTok의 Advantage+)

### Smart+ 진화 (2025→2026)
- **이전:** All-or-nothing 자동화 (전부 또는 없음)
- **2026:** 유연한 선택 - 타겟팅/크리에이티브/예산 독립적 제어
  - 크리에이티브는 자동 테스트, 예산은 가드레일 유지
  - 또는 타겟팅만 자동, 나머지는 수동

### 성숙도별 운영
| 계정 단계 | 자동화 설정 | 이유 |
|---|---|---|
| 초기 | 전체 자동 | 데이터 부족 |
| 성장 | 크리에이티브 자동 + 예산 상한선 | 효율성 + 제어 |
| 성숙 | 수동 + Smart+ 조용히 보조 | 세밀한 최적화 |

## Symphony AI (TikTok 생성 AI 스위트)

### 3가지 통합 기능 (2026년 4월)

1. **Symphony Assistant**: 트렌드 조사, 스크립트, 브레인스토밍
2. **Symphony Creative Studio**: 이미지→비디오, 텍스트→비디오, 상품→동적 비디오, 기존 비디오 변형
3. **자동 투명성 라벨**: 모든 AI 생성 콘텐츠 자동 "AI-generated" 표기

**성과:** 크리에이티브 비용 60–70% 절감, 로테이션 속도 2배.

## TikTok 픽셀 & Events API 이중 구조

### Pixel vs Events API

| | Pixel | Events API |
|---|---|---|
| **전송** | 클라이언트 (브라우저) | 서버 측 |
| **안정성** | 브라우저 환경 의존 | 100% 안정 |
| **권장** | 기본 | 필수 (구매, 회원 이벤트) |

### 표준 이벤트 (8가지)
ViewContent, AddToCart, Purchase, CompleteRegistration, Contact, Download, SubmitForm, Subscribe.

### 앱 계측 (MMP + SKAN)
- **클릭쓰루(CTA):** 클릭 → 행동 추적
- **뷰쓰루(VTA):** 노출만 → 행동 추적
- **MMP 선택:** AppsFlyer, Adjust, Branch 등
- **SKAN 통합:** iOS 설치 속성 추적

## Apple Search Ads (ASA) 4가지 캠프 구조

### 1. Brand Campaigns (방어)
- **목표:** 브랜드명·오타 타게팅
- **매칭:** Exact Match만
- **입찰:** 공격적 (경쟁사 진출 차단)
- **CPA:** 낮음 (이미 의도 있음)

### 2. Category Campaigns (카테고리)
- **목표:** "명상 앱", "수면 음악", "스트레스 해소"
- **매칭:** Broad, Exact 조합
- **입찰:** 중간~높음 (경쟁 심함)
- **광고 포화:** 2026년 증가 추세

### 3. Competitor Campaigns (공략)
- **목표:** 경쟁사 브랜드명 ("Calm", "Headspace")
- **조건:** 우리 앱이 명확한 대체품일 때만
- **입찰:** 공격적 (매우 높은 경쟁)

### 4. Discovery Campaigns (발굴)
- **목표:** 새 키워드 발굴
- **매칭:** Broad Match 또는 Search Match
- **입찰:** 보수적 → 검증 후 인상
- **파이프라인:** 주간 발굴 → Exact Match로 자동 승격 (20+ 설치 조건)

## ASA Custom Product Pages (CPP) 전략

### CPP 정의
- 앱당 최대 35개 맞춤 App Store 페이지
- 스크린샷, 영상, 프로모션 텍스트 재구성

### 성과
- 게임: 전환율 +8%
- 비게임: 전환율 +6.6%

### 실전 운영
- 키워드군별 1개 CPP (예: "fitness" → "운동" 테마)
- A/B 테스트: 스크린샷 순서 2–3개 변형
- 광고 변형과 CPP 1:1 연동

## ASA 키워드 전략 & Discovery → Exact 파이프라인

### 3단계 Tier 구축

```
Tier 1 (High Intent)
└─ Brand + 경쟁사 (Exact Match, 높은 CPM)
└─ 문제 해결 키워드 ("피로 회복", "스트레스 완화")

Tier 2 (Mid Intent)
└─ 카테고리 ("명상 앱", "수면 음악")
└─ 기능 ("향초 추천", "음악 라이브러리")

Tier 3 (Discovery)
└─ Broad/Search Match
└─ 주간 수집 → 성과 검증 → Tier 1 승격
```

### Discovery → Exact 자동 파이프라인

**조건:**
- 최소 20회 설치 + CPA 목표 달성
- 또는 7일 이상 안정적 CVR

**구현:**
- 주간 자동 규칙으로 발견 검색어 → Exact Match로 추가
- 국가/언어별 규칙 분리

**성과:** 6개월 후 발견 키워드 40–60% 우량화.

## ASA 입찰 전략: 수동 vs Maximize Conversions

| 전략 | 특징 | 권장 |
|---|---|---|
| **Maximize Conversions** | 캠프 레벨 CPA 목표 → 동적 입찰 | 초기(월 1–3) |
| **Manual Bidding** | 키워드별 개별 CPM 설정 | 성숙(월 4+) |

**2026 변화:** Apple이 CPA Cap 폐지 예정 (Maximize Conversions 우선).

❌ 상단 퍼널 목표만 (Traffic)

❌ Facebook 기준 4주마다 교체

## 인하우스 실전 노하우 (5가지)

### 1. TikTok = 의도 창조, ASA = 의도 포착
- TikTok에서 발견 → ASA에서 구매 (Full-funnel)
- 따로 운영하면 각각 50% 효율만 남음

### 2. 네이티브 톤은 타협 금지 (TikTok)
- 크리에이티브 비용 > 광고비 (극도 네이티브)
- UGC 외주 → Symphony AI 하이브리드 권장

### 3. 가드레일 위의 자동화
- Smart+ 신뢰하되 예산/CPA 상한선 필수
- 주간 이상 입찰 감시 (급등 대응)

### 4. ASA Discovery 파이프라인은 자동화 가장 수익성
- 주간 발굴 → Exact로 우량화
- 노동 최소, 효율 최고

### 5. Attribution은 불완전함을 받아들이기
- SKAN 24–48h 지연, API 손실 있음
- 그래도 "완전한 데이터" 기다리지 말고 패턴으로 의사결정
